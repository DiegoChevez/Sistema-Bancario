Logica de Programacion****
Un cliente puede registrarse como cliente o dependiente.
//Un cliente es aquel que tiene cuentas bancarias
//Un dependiente es aquella entidad que necesita afiliarse a un banco para realizar retiros o abonos a ciertas cuentas bancarias
Un cliente tiene maximo 3 cuentas bancarias
Un cliente puede ver sus movimientos bancarios: retiros, transferencias a otros usuarios o el mismo banco.
Un dependiente pude realizar retiros y abonos de una cuenta

Una cuenta tiene un Id generado aleatoriamente "random()"

Un cajero puede ingresar clientes y dependientes
Un cajero puede abonar o retirar de una cuenta "Validar por DUI y #Cuenta"
Un cajero puede hacer prestamos a clientes

Cada prestamo tiene un estado
Cada prestamo le prestamo le pertenece a un cliente

Un generente de sucursal puede añadir empleados

Un gerente general puede crear sucursales y añadirles un gerente


Sucursal*
